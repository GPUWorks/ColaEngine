# ColaEngine
这是一个基于OpenGL的光栅化渲染引擎.

## 开发环境
IDE: Visual Studio 2017  
Device: Nvidia GTX1070  
OS: Windows 10  
OpenGL version: 4.0  


## 外部库
[assimp](http://www.assimp.org/)  
[glad](http://glad.dav1d.de/)  
[glfw](http://www.glfw.org/)  
[glm](https://glm.g-truc.net/)  
[stb_image](https://github.com/nothings/stb)  

## 主要功能
* 模型解析
* 骨骼动画
* 网格蒙皮
* 曲面细分球体
* 延迟渲染
* PBR
* IBL
* SSAO
* 泛光
* 景深
